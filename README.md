# Calculator QA Testing Project

## Overview

This project demonstrates basic Quality Assurance practices using Python's unittest framework.

A simple calculator application was developed and tested using automated unit tests.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Exception handling for division by zero

## Technologies

- Python 3
- unittest

## Test Cases

| Test Case | Expected Result |
|------------|----------------|
| Add 5 + 3 | 8 |
| Subtract 10 - 4 | 6 |
| Multiply 3 × 4 | 12 |
| Divide 20 ÷ 4 | 5 |
| Divide by 0 | Raise ValueError |

## How to Run

### Clone Repository

```bash
git clone <repository-url>
```

### Navigate to Project

```bash
cd calculator-qa-project
```

### Run Tests

```bash
python -m unittest test_calculator.py
```
